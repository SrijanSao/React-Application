Initially Configuring the Stealth(core) Environment for Stealth(aware)

Before you can deploy Stealth(aware), you must initially configure the Management Server in your Stealth(core) environment.

This section describes a minimal configuration that enables you install and configure the Management Server so that you can deploy your Solutions from Stealth(aware).

Note: For more detailed information on configuring Stealth(core), see the Unisys Stealth Solution Information Center.
