Security Requirements

The Management Server has the following security requirements:

  • Enable Active Scripting

    Do the following:

    1. Close all open browser windows.

    2. From the Start menu, enter gpedit.msc in the Search box.

    3. In the Local Group Policy Editor window, in the left pane under Computer Configuration, expand Administrative Templates, expand Windows Components, expand Internet Explorer, expand Internet Control Panel, expand Security Page, and then click Internet Zone.

    4. Double-click Allow active scripting.

    5. On the Allow active scripting dialog box, select the Enabled option, and then ensure that Enable appears in the Allow active scripting list (under Options).

    6. Click OK to close the Allow active scripting dialog box.

    7. In the left pane of the Local Group Policy Editor window, select Intranet Zone (under Security Page).

    8. Double-click Allow active scripting.

    9. On the Allow active scripting dialog box, select the Enabled option, and then ensure that Enable appears in the Allow active scripting list (under Options).

    10. Click OK to close the Allow active scripting dialog box.

      (Do not close the Local Group Policy Editor window.)

  • Enable Transport Layer Security (TLS) 1.2

    Do the following:

    1. In the Local Group Policy Editor window, in the left pane, click Advanced Page (which appears under Internet Control Panel that you expanded in the previous procedure).

    2. In the right pane, double-click Turn off encryption support.

    3. In the Turn off encryption support dialog box, select Enabled.

    4. In the Options table, select the value under Secure Protocol combinations, and change the value to Only use TLS 1.2.

    5. Click OK.

    6. Reboot the Management Server.

    7. After rebooting, log on using the user name that you want the Enterprise Manager services to run under, and then do the following to verify that the settings have been applied:

      1. Open Internet Explorer.

      2. On the Tools menu, select Internet options.

      3. On the Internet Options dialog box, select the Advanced tab.

      4. Under Security, verify that the Use TLS 1.2 check box is selected. Verify that all other Use SSL and Use TLS check boxes are cleared.

      5. Click OK to close the Internet Options dialog box.
