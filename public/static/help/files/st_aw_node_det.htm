Modifying Node Details (Node Details Panel)

Use the Node Details panel to view (and modify, if necessary) details for a specific Node, including the classification of the Node and the Node name.

To view the Node Details panel and modify details of a Node, do the following:

  1. Select a Profile, and then click a Node under the Nodes heading.

    Note: You might need to expand the Nodes list to see all Nodes that are members of the Profile.

    The Node Details panel appears.

  2. If necessary, click the classification icon at the top-center of the panel and select a new classification from the menu. The following table provides a description of each classification.

    Notes:

    • If the classification icon is a question mark (?), the Node is unclassified. If you select a classification for an unclassified Node, you cannot change the Node back to unclassified.

    • Stealth(aware) does not automatically classify any Nodes as Admin Users. You can manually classify Nodes as Admin Users, as required.

    Table 5. Node Classifications

    IconDescription

    Database Proxy

    Database Server

    Web Proxy

    Web Server

    Admin User

    Application Server

    Admin Workstation

    Workstation

    User


    Note: The Stealth Management Server, Stealth Standalone Authorization Server, and Stealth Service Mode classifications also appear in this list. Ignore these classifications; they are not supported in this release.

  3. If necessary, in the Node Name field, modify the name of the Node. If the Node is running the Discovery Agent or if the DNS Resolution is set to On in the Project Details panel, the Node Name field is set to the Node host name by default.

    If the Node is not running the Discovery Agent, or if the DNS Resolution is set to Off, or if DNS Resolution is Set to On but the host name cannot be resolved, the Node Name field is set to the Node IP address by default.

    If you want to change the name, it can be a maximum length of 45 characters and must meet the requirements in Supported Characters and Length Restrictions in Enterprise Manager and Stealth(aware).

The following additional information appears on the Node Details panel, which cannot be changed. You can review the following settings:

  • The Host Name field, includes the Node host name, if the host name is known. If the host name is not known, this field is blank.

  • The IP Address field includes the Node IP address.

  • The Parent field includes the name of the Profile that the Node is included in. If you want to view the details of the parent Profile, click the Profile name. See Modifying Profile Details (Profile Details Panel) for more information.

    Note: The icon next to the Profile name indicates the Profile classification.

  • The Flows diagram provides a visual representation of the Channels (and their associated flows) between the current Node and all other Nodes with which it communicates, and includes the following information:

    • The classification and name of the current node.

    • For each flow, either the name of the IANA service or the word Unassigned, a colon, and then the protocol and port.

      Note: If the ports and protocols are not listed on the Internet Assigned Numbers Authority (IANA) registry, they are prefaced by Unassigned. (This includes ports and protocols used by Stealth.)

    • One of the following policies is shown on each flow:

      • Stealth Allow

      • Clear Text Allow

      • Block Clear Text and Stealth

      Note: See Policies for more information.

    • Information about the other Node (with which the selected Node is communicating), including:

      • The IP address, host name, or name of the other Node

      • The classification of the other Node

      • Whether the other Node is providing services to the current Node (the arrow above the classification is pointed up), or the other Node is consuming services from the current Node (the arrow above the classification is pointed down)

      Note: To access the Node Details panel of the other Node, you can click its IP address, host name, or name (depending on what values are defined for the other Node).

Click the X in the upper-right corner or click anywhere on the map to close the Node Details panel.

Note: When you make a change to the Node Details panel, the change is automatically saved.
