Manually Creating Objects (Add Objects Tool)

The Add Objects tool enables you to manually add a new Solution or Profile to the map. This is an optional procedure.

Note: If you set the Affinity Level to Off, you cannot use this tool.

To use the Add Objects tool, do the following:

  1. Click the Add Objects tool.

             

    The Add Objects panel appears.

  2. From the Add Objects panel, drag a Solution or Profile and drop it onto the map.

  3. Enter a unique name for each Solution or Profile. This name can be a maximum length of 45 characters and must meet the requirements that are described in Supported Characters and Length Restrictions in Enterprise Manager and Stealth(aware).

  4. Optionally configure the following additional values for Solutions or Profiles. (You can configure these values at a later time, if you choose.)

    • For new Solutions, optionally add Profiles to the Solution and set the Stealth Enabled value. See Modifying Solution Details (Solution Details Panel) for more information.

      Note: If you change the Affinity Level, any empty Solutions in the model are deleted. To avoid this, add at least one Profile to the empty Solution.

    • For new Profiles, optionally configure the Group Name and set the Stealth Enabled value. See Modifying Profile Details (Profile Details Panel) for more information.

  5. Repeat the previous steps to add as many Solutions and Profiles to the map as required.

  6. When you are done adding new Solutions or Profiles, click the X in the corner to close the Add Objects panel.

  7. Manually sort the new Solution or Profile you created.

For more information about Solutions, Profiles, Channels, and the Stealth(aware) Model, see Stealth(aware) Map and Components.
