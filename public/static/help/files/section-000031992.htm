Upgrade Procedure

 Caution

If you are upgrading from Stealth(aware) version 3.2.004 or 3.2.010, perform the steps in Upgrade Prerequisite for Stealth(aware) Version 3.2.004 and 3.2.010 before performing this procedure; otherwise, all data will be removed from the database and any projects you previously created projects will be deleted.

For Stealth(aware) version 3.2.0.11 or later, no prerequisite steps are required.

To upgrade from your current version of Stealth(aware) to a new version, do the following:

  1. If you have not already done so, do the following:

    1. Save the new version of the Unisys Stealth(aware) Release ISO to the system running Stealth(aware).

    2. Mount the ISO file or extract the ISO file.

  2. On the new ISO file, double-click the Stealth-aware_Installer.exe file.

    The new version of Stealth(aware) is installed and automatically launched.

  3. Close Stealth(aware).

  4. Launch Stealth(aware) from the icon on the desktop or by using the Start menu.

  5. On the Stealth(aware) Start Screen, verify that the software version on the bottom of the screen has been updated, and verify that the Last Opened Project option includes the name of your most recently opened project. This indicates that Stealth(aware) version has been upgraded and your database has been successfully migrated.

Note the following:

  • If you upgraded from Stealth(aware) version 3.2.004 or 3.2.010, the previous version of the database is saved as described in Upgrade Prerequisite for Stealth(aware) Version 3.2.004 and 3.2.010.

  • If you upgraded from Stealth(aware) 3.2.011 or later, the previous version of the database is saved as insight_<revision>_<yyyymmdd>_<hhmmss>.db in the C:\Users\<user name>\AppData\Roaming\Stealth folder.
