Enabling Stealth on Windows Endpoints

The Stealth Dashboard provides a method for you to enable and disable the Stealth service, if this option is allowed in your environment. (You might not be permitted to disable Stealth.) The Stealth Dashboard also provides status information on your current Stealth configuration.

You can access the Stealth Dashboard by clicking the Stealth Shield icon in the task bar. If the Dashboard does not appear in the task bar, you can access it from the Start menu by typing Stealth Dashboard in the Search box.

The following options are provided on the Main page of the Stealth Dashboard, depending on whether Stealth is configured as Always On or On Demand, as follows:

  • If Stealth is configured as Always On, the configurations are automatically contacted in the order in which they were configured, until a connection is established or until there are no more configurations available. (You cannot disable Stealth.)

  • If Stealth is configured as On Demand

    • If you have more than one configuration, ensure that the configuration you want to connect to is shown in the Connect to list. If you cannot connect to that configuration, you can select a different configuration that you want to connect to.

      Click Connect to connect to the selected configuration.

    • If you have only one configuration, click Connect to connect to that configuration.

For more information, click Help.
