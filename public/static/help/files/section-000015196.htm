Optionally Disabling Stealth on Selected Network Adapters

Perform one of the following procedures to disable Stealth on network adapters.

Disabling Stealth on Selected Network Adapters (Before Endpoint Software Installation or Upgrade)

By default, Stealth is enabled on all network adapters on the Windows endpoint when Stealth endpoint software is installed or upgraded (excepting known virtual adapters that Stealth does not bind to, such as VPN adapters). If the system has multiple network adapters, you can disable Stealth for selected adapters before the endpoint installation or upgrade using the Stealth Network Adapter Exclusion Utility.

When you run this utility, a new registry key is written to the Windows Endpoint Registry, and the changes are applied during the next Stealth endpoint software installation or upgrade to a later version of Stealth endpoint software. You can rerun this utility at any time to update the registry.

Note: If you run this utility to make changes to the registry entry, and then install the same version of endpoint software again, your changes will not take effect. You must uninstall and reinstall the Stealth software or upgrade to a later version of endpoint software to apply the changes from the registry.

These changes persist when the endpoint is rebooted and if the endpoint software is upgraded or uninstalled.

Do the following:

  1. On Unisys Stealth Solution Release 3.x ISO file, navigate to the support directory, and copy the Stealth Network Adapter Exclusion Utility file, which is named ExcludeAdapters.exe.

  2. Save this file to your endpoint, and run it on the endpoint by double-clicking ExcludeAdapters.exe.

  3. In the Network Adapters Available to Stealth list, select a network adapter that you want to exclude, and then click Exclude.

    Note: If any network adapters that you want to exclude do not appear in the list, close the utility, ensure that the network adapters are connected and enabled in Windows, and then reopen the utility.

  4. Repeat the previous step to exclude as many network adapters as needed.

  5. In the Stealth Adapter Exclusion List, ensure that all of the network adapters you want to exclude are listed.

    If there are any network adapters that appear in this list that you want to be Stealth enabled, you can remove them.

  6. Click Save.

    A new registry entry is written to HKEY_LOCAL_MACHINE\SOFTWARE\Unisys\MLSTPGW named ExcludeDeviceNames.

Updating Selected Network Adapters (After Endpoint Software Installation)

Alternatively, after the endpoint software is installed, you can disable Stealth for selected adapters by doing the following:

  1. Access Control Panel, and select Network and Sharing Center.

  2. Access the Local Area Connection Properties for the network adapter for which you want to disable Stealth.

  3. On the properties dialog box, clear the "Unisys Stealth Solution" check box, and then click OK.
