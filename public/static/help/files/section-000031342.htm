Updating the Discovery Agent Settings on a Windows Node

To update the Discover Agent, do the following on a Windows Node:

  1. From the Start menu, enter Notepad in the Search box.

  2. Right-click Notepad, and select Run as administrator.

  3. If you are asked if you want to allow Notepad to make changes to this computer, click Yes.

  4. In Notepad, select the File menu, and then click Open.

  5. On the Open dialog box, change the file type from Text Documents (*.txt) to All Files.

  6. Browse to the C:\Program Files\Unisys\Stealth Solution\Discovery directory and open settings.xml.

    Note: Be sure to open the settings.xml file in the Discovery directory. Your Node might include other settings.xml files that must not be changed.

  7. As required, update the values as follows.

     Caution

    Be very careful when updating these settings. If you make an error, you might prevent the Discovery Agent from operating on the Node. In this case, you must uninstall the Discovery Agent using the standard Windows method and reinstall it using the directions in Installing the Discovery Agent on Nodes Running Windows.

    • If you want to update the Windows file share you configured in Configuring a Windows File Share for the Discovery Agent Network Traffic Data Files to use a new Windows file share with the same password, you can update the following line.

      Note: The password you enter for the Windows file share is encrypted and cannot be changed. You can only update the file share if the new file share includes a user with the same password that you entered when you installed the Discovery Agent. If you want to change the Windows file share and use different credentials, you must reinstall the Discovery Agent software.

      <remote method="FILE" user="user name" 
     pass="encrypted password">\\share\folder</remote>

    • If you want to update how often (in seconds) you would like to start a capture, update the following value (where the default is 840):

      collect start="840"

    • If you want to update how long (in seconds) you would like each capture to last, update the following value (where the default is 65):

      length="65"

    • If you want to update how often (in seconds) you would like to move completed capture files to the remote directory, update the following value (where the default is 1800):

      push="1800"

  8. From the File menu, select Save As.

  9. In the Save As dialog box, set the Save as type to All Files, enter the file name settings.xml, and then click Save.

  10. If you are prompted for encoding, accept the default.

  11. Close Notepad.

  12. Do the following to restart the Unisys Stealth Discovery service:

    1. From the Start menu, enter Services in the Search box, right-click Services, and select Run as administrator.

    2. Locate the Unisys Stealth Discovery service in the list of Services.

    3. Right-click the Unisys Stealth Discovery service and select Restart.
