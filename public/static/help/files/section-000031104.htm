Collecting Diagnostics for Stealth(aware)

To report an issue to Unisys, you use a User Communication Form (UCF). To report an issue for Stealth(aware), create a UCF and attach the following files from the system running Stealth(aware):

  • The insight.db file in C:\Users\<user name>\AppData\Roaming\Stealth

  • The following files in C:\Users\<user name>\AppData\Roaming\Stealth\logs:

    • stealth-aware.log

    • stealth-aware-ui.log
