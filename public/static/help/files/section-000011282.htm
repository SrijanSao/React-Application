Enabling or Disabling Stealth Modes and Stealth for Linux Endpoints

Enabling or Disabling Stealth on Boot

The Stealth service is configured to start when the system boots. To prevent the service from starting at boot, enter the following command:

  • For Red Hat Enterprise Linux and SUSE Linux Enterprise Server 11.x, enter the following command:

    chkconfig stealthd off

  • For SUSE Linux Enterprise Server 12.x, enter the following command:

    systemctl disable stealthd

  • For Ubuntu Linux, enter the following command:

    update-rc.d stealthd disable

Enabling or Disabling the Stealth Service

  • To disable Stealth on a Red Hat Enterprise Linux, SUSE Linux Enterprise Server 11.x, or Ubuntu Linux system, enter the following command:

    service stealthd stop

    To enable Stealth on a Red Hat Enterprise Linux, SUSE Linux Enterprise Server 11.x, or Ubuntu Linux system, enter the following command:

    service stealthd start

  • To disable Stealth on a SUSE Linux Enterprise Server 12.x system, enter the following command:

    systemctl stop stealthd

    To enable Stealth on a SUSE Linux Enterprise Server 12.x system, enter the following command:

    systemctl start stealthd
