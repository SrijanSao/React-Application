Configuring a New Password and Configuring Licensing in Enterprise Manager

Do the following to configure a new password for the portaladmin account and to configure Stealth licensing in Enterprise Manager:

  1. If you have not already done so, launch Enterprise Manager by double-clicking the Unisys Stealth Enterprise Management Portal shortcut on the Management Server desktop.

  2. Sign in to Enterprise Manager with the Portal Administrator account credentials. (The default user name is portaladmin and the default password is AdminPortal1#.)

  3. Change the default portaladmin user account password by doing the following.

     Caution

    You must record this new password. It cannot be recovered.

    1. Click Change Password in the upper-right corner of the page.

    2. On the Change Password dialog box, enter the default password (AdminPortal1#) in the Old Password box.

    3. Enter the new password in the New Password and Confirm New Password boxes.

      Note: The password must be between six and 50 characters, and it must include all of the following:

      • At least one uppercase letter

      • At least one lowercase letter

      • At least one number

      • At least one of the following special characters:

        ! @ # $ % ^ & * ( ) _ + =

    4. Click Save.

    5. Click Sign Out to sign out of Enterprise Manager, and then sign back in using the new password.

  4. Install the local license file by doing the following.

    Notes:

    • The Management Server can have only one license file installed. See the Unisys Stealth Solution Information Center for more information on licensing.

    • Ensure that you have the Welcome Letter that you received with your order; you are instructed to enter the SCN that is provided in the Welcome Letter. (If you do not have the Welcome Letter, you can install temporary licenses.)

    • You can install temporary licenses in order to test your configuration. This temporary licenses provide 25 Client licenses, 25 Server licenses, 25 Mobile licenses (for remote users using SRA clients), and 25 VGD licenses (Virtual Gateway Device licenses for Clear Text endpoints that use the Secure Virtual Gateway) for 90 days. (However, note that Mobile licenses and VDG licenses cannot be used in conjunction with an environment configured using Stealth(aware).If you want to use these temporary licenses, perform the steps for temporary licenses in the following procedure.

    1. Select Manage from the menu bar.

    2. Select the License tab.

    3. Click Install SCN at the bottom of the page.

    4. On the Install SCN dialog box, enter the SCN that was provided in your Welcome Letter in the SCN box.

      Note: If you want to use temporary licenses, you can enter the SCN from the Welcome Letter, or you can enter any numeric string in the SCN box (for example, 1234567890). You can enter any number, but it must be unique within the configuration, and it cannot match another temporary SCN.

    5. Ensure that the Management Server is selected in the System list.

    6. Click Save.

      Enterprise Manager installs the SCN. This process can take several minutes to complete.

    7. Click the Refresh icon at the bottom of the License table to refresh the table and display the SCN that you installed.

      After you install the SCN, a file with an encrypted license certificate is generated on the Management Server as a .blo file. The .blo file is located in the C:\Stealth Files\Certificates\ directory and is named according to your SCN.

  5. To install temporary licenses do the following.

    Note: To install permanent licenses, skip to the next step.

    1. Select the SCN row in the table.

    2. Click Update License.

    3. Enter Test in the License Key box and enter Licenses in the Verification Key box.

    4. Click Save.

      Skip to the final step in this procedure.

  6. To install permanent licenses, do the following:

    1. Log on to the Unisys Download Center as described in your Welcome Letter.

      The Download Center Home page appears and displays your corporate contact information.

    2. Select Download Management in the left pane.

    3. In the Order Number box, enter the order number provided in your Welcome Letter, and click Submit.

    4. On the Download Management page, under the Product Management heading, click the Stealth Software Licensing link that corresponds to your order number.

    5. On the Stealth Software Licensing page, select the SCN that you installed on the Management Server from the SCN Number list and click Submit.

    6. Click Browse next to the Encryption Certificate box, and browse to the .blo file that was generated by Enterprise Manager. This file was created in the C:\Stealth Files\Certificates\ directory and is named according to your SCN.

    7. Click Update.

      The Download Center provides a license file, which is an encrypted text file that contains your license keys.

    8. Save and open the license file.

    9. On the Enterprise Manager License tab, select the SCN for the license you are installing and click Update License.

    10. On the Update License dialog box, copy the strings from the license file in the License Key box and Verification Key box.

      Note: If you open the text file in Notepad, it is recommended that you enable Word Wrap (by clicking Format and selecting Word Wrap) to ensure that you copy the full text of each string.

    11. Click Save.

      Enterprise Manager installs the licenses. This process can take several minutes.

  7. Click the Refresh icon at the bottom of the License table to refresh the display with the updated license information. The Total License column displays the updated license count and the Status displays Installed Locally.

    The licenses are installed and ready for use.
