Understanding How Long the Stealth Discovery Agent Should Run

Allowing the Discovery Agent to collect too much data can cause significant delays when the data is imported and analyzed; therefore, it is important to deploy the Agent strategically and for short periods of time, as follows:

  • Recommended short-term capture period values (12 hours or less)

    Note: The following are the default values when you install the Discovery Agent.

    • How often to start a capture: 840 seconds (14 minutes)

    • How long to capture data: 65 seconds (1 minute, 5 seconds)

    • How often to move files: 1,800 seconds (30 minutes)

  • Recommended long-term capture period values (several days)

    • How often to start a capture: 3,240 seconds (54 minutes)

    • How long to capture data: 120 seconds (2 minutes)

    • How often to move files: 3,600 seconds (60 minutes)
