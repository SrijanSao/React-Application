Backing Up the Stealth(aware) Database

It is recommended that you back up the Stealth(aware) database on a weekly basis or before making any significant changes to the database (for example, before importing new traffic data files or before using the Create Solutions tool). To back up the database, do the following:

  1. Navigate to C:\Users\<user name>\AppData\Roaming\Stealth folder.

  2. Copy the insight.db file to a secure location.

    It is a best practice to rename this file in the secure location to append the date and time.

     Caution

    Do not change the name of the insight.db file in the C:\Users\<user name>\AppData\Roaming\Stealth folder.

If you need to restore this file, do the following:

  1. Ensure that Stealth(aware) is closed.

  2. Copy the file from the secure location in which you saved it, and paste it in C:\Users\<user name>\AppData\Roaming\Stealth.

  3. Rename the file as insight.db.
