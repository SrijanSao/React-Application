Updating the Discover Agent Settings on a Linux Node

To update the Discover Agent, do the following on a Linux Node:

  1. Log on to the system as root, and then open a terminal window.

  2. Enter the following command to update the settings:

    nodediscoveryd-conf

  3. When you are asked where you want to move capture files, enter the location of the mount point you created for the remote share. Press Enter to retain the previous value you entered. Alternatively, enter a new value.

  4. When you are asked how often (in seconds) you would like to start a capture, press Enter to retain the previous value you entered. Alternatively, enter a new value.

  5. When you are asked how long (in seconds) you would like each capture to last, press Enter to retain the previous value you entered. Alternatively, enter a new value.

  6. When you are asked how often (in seconds) you would like to move completed capture files to the remote share, press Enter to retain the previous value you entered. Alternatively, enter a new value.

  7. Enter one of the following commands to restart the Discovery service, based on your operating system:

    • For Red Hat Enterprise Linux 6.x and SUSE Linux Enterprise Server 11.x, enter the following command:

      service nodediscoveryd restart

    • For Red Hat Enterprise Linux 7.x and SUSE Linux Enterprise Server 12.x, enter the following command:

      systemctl restart nodediscoveryd

    • For Ubuntu, enter the following command:

      restart nodediscoveryd
