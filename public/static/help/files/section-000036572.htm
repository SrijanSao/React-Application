Upgrade Prerequisite for Stealth(aware) Version 3.2.004 and 3.2.010

If you are currently running Stealth(aware) version 3.2.004 or 3.2.010, perform the following prerequisite steps before performing the upgrade procedure. The Stealth(aware) software version is listed on the bottom of the Start Screen when you initially launch Stealth(aware).

If you are running a later version of Stealth(aware) software, skip to Upgrade Procedure.

 Caution

If you do not perform these steps, upgrading from Stealth(aware) version 3.2.004 or 3.2.010 will remove all data from the database and permanently delete any projects that you previously created.

Do the following:

  1. Save the new version of the Unisys Stealth(aware) Release ISO to the system running Stealth(aware).

  2. Mount the ISO file or extract the ISO file.

  3. On the ISO file, navigate to the upgrade directory.

  4. Copy the Run_Stealth-aware_upgrade.ps1 file from the ISO upgrade directory to any location on your system; for example, save it to C:\Windows\temp.

  5. Open a PowerShell window and enter the following command:

    cd "<directory>"

    For example, enter cd “C:\Windows\temp”.

  6. Enter the following command:

    .\Run_Stealth-aware_upgrade.ps1

    You receive a message that the database has been successfully saved and you must install the new version of Stealth(aware).

  7. Close the PowerShell window.

  8. Navigate to C:\Users\<user name>\AppData\Local\Stealth-<mm>-<dd>-<yyyy>_<hh>_<mm>_<ss>\app-3.2.0, and locate the insight.db database file.

  9. Copy the insight.db database file from this folder, and paste it into the C:\Users\<user name>\AppData\Roaming\Stealth folder.
