Installing Stealth Endpoint Software on Windows

Do the following to install the Stealth endpoint software on Windows endpoints on all systems running supported Windows operating systems in the environment that you want to Stealth-enable. After you meet the prerequisites, you can either perform the procedure in Manual Installation Procedure (Non-Silent) or Silent Installation Procedure.

Note: For licensing purposes, you should install the endpoint software first on the Management Server before installing it on other Windows endpoints.
