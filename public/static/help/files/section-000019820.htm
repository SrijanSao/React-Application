Prerequisites

Before you begin installing the Stealth endpoint software on Windows endpoints, do the following:

  1. Ensure that you met all of the Windows endpoint requirements in Windows Endpoint Requirements.

  2. Ensure that the trusted root certificate (and any required intermediate certificates) have been imported to the Windows endpoint, as described in Importing the Trust Chain for the Endpoint Package Signing Certificate on Windows Endpoints.

 Caution

If you do not meet these prerequisites, the Stealth endpoint software cannot be installed. The installation process will terminate and you will receive an error code that describes the problem. Additional information on the error will be available in the Windows Application Event Log.
