Discovery Agent Supported Operating Systems

There is one Discovery Agent for Windows and one Discovery Agent for Linux. The Discovery Agent can be installed on all systems that support Stealth(core).

The following operating systems are supported:

  • Windows operating systems:

    • Windows 7

    • Windows Server 2008 R2

    • Windows 8 and Windows 8.1

    • Windows Server 2012 and Windows Server 2012 R2

    • Windows 10

  • Linux operating systems:

    • Red Hat Enterprise Linux (RHEL) 6.x, 32-bit and 64-bit

    • Red Hat Enterprise Linux (RHEL) 7.x, 64-bit

    • SUSE Linux Enterprise Server (SLES) 11.x, 32-bit and 64-bit

    • SUSE Linux Enterprise Server (SLES) 12.x, 64-bit

    • Ubuntu 12.04 LTS, 32-bit and 64-bit

    • Ubuntu 14.04 LTS, 64-bit
