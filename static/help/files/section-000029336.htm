System and ISO Requirements

You can install the Management Server software on the same system that you used to install Stealth(aware), as long as that system meets the requirements for the Management Server software. Alternatively, you can install the Management Server software on a separate system in your environment.

The Management Server software can be installed on a virtual machine or a physical server. The following are the system requirements:

  • The software must be installed on a server that is dedicated to running the Management Server software and, optionally, Stealth(aware). The server should not be used for any other purpose in the datacenter.

  • The Management Server must run Windows Server 2012 R2 x64 (any version).

    Be sure to configure Server with a GUI mode. See https://technet.microsoft.com/en-us/library/hh831786(v=ws.11).aspx for more information.

  • Apply all available cumulative and security updates provided for your Microsoft operating system.

  • The Management Server host name must not include an underscore. (All other Windows special characters are supported.)

  • The Management Server must have a static IPv4 address.

  • The Management Server must meet the following hardware requirements:

In addition, ensure that you have the Unisys Stealth Solution Release ISO file, which contains the software required for the Management Server installation. You can download this file as directed in the Welcome Letter you receive from Unisys. Save the ISO file on the system you want to act as the Management Server.
