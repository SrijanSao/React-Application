Installing Stealth(aware)

 Caution

Do not use this procedure to reinstall or upgrade Stealth(aware) on a system that it is already installed on. Depending on the currently installed version of Stealth(aware), using this procedure might remove all data from the database and delete any projects that you previously created.

If you need to upgrade to a new version of Stealth(aware), see Upgrading Stealth(Aware).

To install Stealth(aware), do the following:

  1. Download the Unisys Stealth(aware) Release ISO as directed in the Welcome Letter you received from Unisys.

    Note: If you are an existing Stealth customer, you can access the Stealth(aware) software from the Unisys Product Support website, on the Stealth Solution webpage. However, note that using Stealth(aware) in an existing Stealth environment is not supported in this release, and there is no upgrade path that includes use of Stealth(aware). See Configuration Requirements for Integrating with Stealth(core) for more information.

  2. Save the ISO file on the system on which you want to run Stealth(aware).

  3. Mount the ISO file or extract the ISO file.

  4. Double-click the Stealth-aware_Installer.exe file from the ISO file to launch the installer.

The software is installed to the C:\Users\<user name>\AppData\Local\Stealth\app-<version> directory. In addition, a new item is added to the Start menu, and a new shortcut named Stealth is created on the desktop to enable you to access Stealth(aware).
