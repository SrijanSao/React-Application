Upgrading the Discovery Agent on Windows Nodes

To upgrade the Discovery Agent running on a Windows Node, perform the procedure in Installing the Discovery Agent on Nodes Running Windows. Your previous settings for the Remote Share, Remote Share User, Remote Share Password are automatically retained, as are your previous capture values. However, you can change any of these settings, if required.

The previous version of the Discovery Agent software is uninstalled, and the new version is installed.
