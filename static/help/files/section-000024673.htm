Creating a New Project

To create a new project, on the Start screen, click New Project.

The map appears and looks like the following:

Figure 3. Stealth(aware) Map: New Project


The Stealth(aware) map for a new project provides the following:

  • Project Details panel – Provides details about the Project. See Updating the Project Name and Details (Project Details Panel).

  • Modeling Tools – Tools to build a Stealth(aware) Model. See Using Stealth(aware) to Create a Model.

  • ? – Opens the online version of this document.

To change your view, do any of the following:

  • Pan across the map to view different objects

  • Use the scroll wheel on your mouse to zoom in on objects

  • Use the Zoom In and Zoom Out buttons in the bottom-left corner of the map
