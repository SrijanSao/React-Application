Software Installed During the Installation Process

The following software will be installed during the installation process.

  • Windows .NET Framework version 4.6.1

    Note: If .NET Framework is already installed, the Stealth Enterprise Manager setup process checks the current version. .NET Framework version 4.6.1 is only installed if that version is not already present.

  • Visual C++ 2015 and 2010 (x86 and x64) Redistributables

  • Java™ SE Runtime Environment Version 8.0 (x64)

  • MySQL (MySQLVersion 5.6.27 and MySQL Connector/ODBC 5.3.4)

  • Liferay Portal 6.2, Tomcat Server 7.0.62 (for Enterprise Manager), Tomcat Server 8.0.36 (for Stealth EcoAPI), and Liferay (as a Windows service) 8.0.5

    Notes:

    • The Management Server must not be running any other version of Tomcat besides the version installed with the Enterprise Manager software. If the system on which you plan to install the Management Server is already running Tomcat, you must remove it before beginning the Enterprise Manager software installation.

    • Liferay Portal and Tomcat Server are installed by the Unisys LiferayTomcatService.msi installer, which has a Stealth Solution 3.x build version.

    • Tomcat Server runs as a Windows service, which is named Stealth Enterprise Manager. During the installation procedure, you are prompted to create a user name and password for this service. The default user name is Tomcatuser, but it can be changed to another unique user name in the environment that meets Windows user account naming conventions. The installation procedure creates a non-administrator user account configured to Log on as a service.

  • Pegasus (cimserver.exe) 2.12.0.0

    Note: Pegasus is installed as part of the Unisys Stealth Enterprise Manager installer.
