Updating Configurations Using the StealthConfiguration Utility

 Caution

In the following procedures, you must spell and capitalize all commands exactly as written.

Therefore, you might want to do the following:

  1. Copy-and-paste these commands from this document into a text editor.

  2. Update the variables for your environment.

  3. Remove the line breaks.

  4. Paste the resulting command into the command prompt window by right-clicking and selecting Paste.

As described in Using the StealthConfiguration Utility to Create Required Configurations, you use the StealthConfiguration Utility to create a configuration for the Management Server and for the endpoints. If you need to create a new endpoint configuration or a new Administrative configuration, perform one of the procedures in this topic.
