Moving Nodes Between Profiles and Moving Profiles Between Solutions

Moving Nodes Between Profiles

If Affinitization is set to High, Medium, or Low, each Node is included in a Profile on the map. To remove a Node from a Profile, use the Profile Details panel for the Profile that the Node is included in. From the Profile Details panel, from the Nodes list, drag-and-drop the Node onto the map. The Node is then included in a new Profile. For more information, see Modifying Profile Details (Profile Details Panel).

To move a Node into a new Profile, drag the Profile that includes the Node on top of another Profile to combine the Profiles and member Nodes into one Profile, and then click Yes, Continue. The Profile that you drag is merged into the Profile on which you drop it, which means that the “dragged” Profile takes on the settings, policies, and name of the other Profile.

When you add or remove Nodes from Profiles, the classification of the Profile might be automatically updated. For example, if you add a Node classified as a Workstation to a Profile that originally included only Nodes classified as Databases, that Profile classification would change from Database to Mixed. If you then removed the Node classified as a Workstation, the Profile would return to being classified as Database.

Moving Profiles Between Solutions

On the map, you can drag Profiles into closed Solutions or into expanded Solutions. In addition, you can remove a Profile from a Solution by expanding the Solution and dragging out the Profile, or by using the Solution Details panel for the Solution that the Profile belongs to. From the Solution Details panel, from the Profiles list, drag-and-drop the Profile it onto the map. For more information, see Modifying Solution Details (Solution Details Panel).
