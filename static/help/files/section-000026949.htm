Launching Stealth(aware)

To launch Stealth(aware), double-click the Stealth shortcut on the desktop of the system running Stealth(aware).

Stealth(aware) is launched, and the Stealth(aware) Start screen appears like the following:

Figure 5. Stealth(aware) Start Screen


From the Start screen, you see the following options:

  • Create a new project

  • Open your last opened project

  • Open a saved project from the Project Menu (in the upper left)

  • View the Help by clicking the ? icon (in the upper right)
