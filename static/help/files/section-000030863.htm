Uninstalling Stealth(aware)

If you need to uninstall Stealth(aware), use the standard Windows method to uninstall a program. Stealth(aware) is listed as “Stealth” under Windows Programs and Features. The Stealth(aware) application and your associated project data will be deleted from the system.

In addition, use Windows Explorer to navigate to C:\Users\<user name>\AppData\Local, and then delete the Stealth directory.
