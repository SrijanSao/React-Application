Modeling New Environments

If you are modeling a different network segment in the same environment, or if you are modeling a different environment, it is recommended that you create a new database file as follows:

  1. Close Stealth(aware).

  2. Use Windows Explorer to navigate to C:\Users\<user name>\AppData\Roaming\Stealth and locate the insight.db database file.

  3. Rename insight.db to use a new name, for example, insight.db_<date> or insight.db_<client name>.

  4. Launch Stealth(aware), and verify that the only available option is Create a New Project, as shown in Stealth(aware) Start Screen.

This action will create a new insight.db database file. In this way, data is isolated and performance is maintained.

If you want to view the previous database file in Stealth(aware), repeat the previous steps, and rename the previous database file insight.db.
