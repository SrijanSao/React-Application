Stealth(aware) System Requirements

 Caution

It is highly recommended that you install Stealth(aware) on a physical system to which you have a direct monitor connection. If you use Windows Remote Desktop or VMware console to access the system running this software, graphics performance will be significantly impacted. In addition, the time required to import traffic data files could be significantly increased if you install Stealth(aware) on a virtual machine.

You can install Stealth(aware) on any of the following:

  • A Windows workstation running Windows 7 x64, Windows 8.1 x64, or Windows 10 x64

  • A Windows server that you want to act as the Stealth Management Server, running Windows Server 2012 R2 x64 (any version)

    Note: For Windows Server 2012 R2 x64, be sure to configure Server with a GUI mode. See https://technet.microsoft.com/en-us/library/hh831786(v=ws.11).aspx for more information.

The following are the minimum system requirements for Stealth(aware):

Best Practice: Use system graphics that support hardware acceleration.
