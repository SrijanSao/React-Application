Windows Endpoint Requirements

The following are supported Windows operating systems (all versions):

  • Windows 7

  • Windows Server 2008 R2

    Note: Windows Server 2008 R2 Clustering is also supported.

  • Windows 8 and Windows 8.1

  • Windows Server 2012 and Windows Server 2012 R2

  • Windows 10

In addition, all Windows endpoints meet the following requirements:

  • .NET Framework version 3.5 with Service Pack 1 or .NET Framework version 4.x

    Note: Verify that the version of .NET Framework that you want to use is installed and activated using Control Panel, Programs and Features, or using Server Manager Dashboard (depending on the Windows operating system).

  • For Windows 7 and Windows Server 2008 R2 only: Verify that the following Windows security update and hotfix patches are installed.

    • “Availability of SHA-2 Code Signing Support for Windows 7 and Windows Server 2008 R2 (KB3033929)” – Included in Windows Update and available at https://www.microsoft.com/en-us/download/details.aspx?id=46148

    • “The Untrusted publisher dialog box appears when you install a driver in Windows 7 or Windows Server 2008 R2 (KB2921916)” – Windows hotfix included on the Unisys Stealth Solution Release ISO file as Windows6.1KB2921916-<x64 or x86>.msu

    • “Kernel-Mode Driver Framework version 1.11 update for Windows Vista, Windows Server 2008, Windows 7, and Windows Server 2008 R2 (KB2685811)” – Included in Windows Update and available at https://www.microsoft.com/en-us/download/details.aspx?id=38423

    Note: If you install any of these security updates, you must reboot your endpoint before continuing with the Stealth software installation.

In addition, you must also meet the memory requirements for the specific Windows operating system.
