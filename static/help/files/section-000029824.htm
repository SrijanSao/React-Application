Deleting the Administrative Configuration, the Endpoint Configuration, and Creating New Configurations

 Caution

This is an advanced topic that requires extensive use of Enterprise Manager. Refer to the Unisys Stealth Solution Information Center for more information about performing any of the steps in this procedure.

If you need to delete the Administrative configuration and endpoint configuration you created using the StealthConfiguration Utility, you can uninstall and reinstall the Enterprise Management software from the Management Server. Alternatively, you can perform the following procedure.

When you perform this procedure, you delete all of the components you created when you ran the StealthConfiguration Utility, including the Administrative configuration, the endpoint configuration, and the Stealth endpoint packages. After you perform this procedure, you must redeploy using Stealth(aware) and install new versions of the Stealth endpoint packages on the Management Server and on all endpoints.

Do the following:

  1. Ensure that you are logged on to the Management Server as a local administrator user.

  2. Open a command prompt using the Run as administrator option.

  3. Enter the following command to access the StealthConfiguration Utility directory:

    cd "C:\Program Files\Unisys\EnterpriseMgmt\Stealth Manager"

  4. Enter the following command to delete the endpoint configuration:

    StealthConfiguration.exe -v DeleteProject="<Endpoint Configuration Name>"

    Notes:

    • Ensure that you spell and capitalize this command exactly as written.

    • The -v option enables verbose mode and provides additional feedback on the results of the command. You should see feedback as the project is deleted.

    For example, enter StealthConfiguration.exe -v DeleteProject="AwareEndpoints".

  5. Launch Enterprise Manager, and log in using portaladministrator credentials.

  6. Do the following to delete the existing endpoint packages:

    1. Select Manage from the menu bar.

    2. On the Package tab, on the Endpoint Package subtab, press Ctrl and select all of the endpoint packages on the page.

    3. At the bottom of the page, click Delete.

    4. Click Yes to confirm the deletion.

    5. Wait until the endpoint packages are deleted.

  7. Do the following to delete the Administrative configuration:

    1. Select Configure from the menu bar, and then select the Configuration tab.

    2. Select the Administrative configuration (for example, AwareAdmin).

    3. At the bottom of the page, click Delete.

    4. Click Yes to confirm the deletion.

    5. Wait until the configuration is deleted.

  8. Do the following to delete the associated COIs:

    1. On the Configure page, select the COI tab.

    2. Select the first COI associated with the Administrative configuration (for example, AwareAdminLicenseCOI).

    3. At the bottom of the page, click Delete.

    4. Click Yes to confirm the deletion.

    5. Wait until the COI is deleted.

    6. Select the second COI associated with the Administrative configuration (for example, AwareAdminServiceCOI).

    7. At the bottom of the page, click Delete.

    8. Click Yes to confirm the deletion.

    9. Wait until the COI is deleted.

  9. Optionally, do the following to delete the clear text filters you created.

    Note: This step is optional, because you can create an unlimited number of clear text filters that use unique names. If you want to delete any clear text filters, be sure that you delete all associated filter sets, lists, and qualifiers.

    1. Select Filters from the menu bar, and then select Filter Set from the Filter tree.

    2. Select the first filter set on the right side of the page.

    3. At the bottom of the page, click Delete.

    4. Click Yes to confirm the deletion.

    5. Repeat the previous two steps to delete all additional filter sets.

    6. Select Filter List from the Filter tree.

    7. Select the first filter list on the right side of the page.

    8. At the bottom of the page, click Delete.

    9. Click Yes to confirm the deletion.

    10. Repeat the previous two steps to delete all additional filter lists.

    11. Select Qualifier from the Filter tree.

    12. On the Protocol/Port tab on the right side of the page, select the first qualifier in the list.

    13. At the bottom of the page, click Delete.

    14. Click Yes to confirm the deletion.

    15. Repeat the previous two steps to delete all additional qualifiers on the Protocol/Port tab.

    16. On the IP on the right side of the page, select the first qualifier in the list.

    17. At the bottom of the page, click Delete.

    18. Click Yes to confirm the deletion.

    19. Repeat the previous two steps to delete all additional qualifiers on the IP tab.

  10. Close Enterprise Manager.

  11. Perform the steps in Using the StealthConfiguration Utility to Create Required Configurations.

  12. Perform the steps in Installing the Stealth Endpoint Package on the Management Server.

  13. Perform the steps in Deploying the Stealth(aware) Model.

  14. Perform the steps in Installing Stealth Endpoint Software.
