Restoring a Previous Version of Stealth(aware)

If you upgraded Stealth(aware), but you need to restore a previous version due to an issue with the database upgrade or due to an upgrade software error, you can perform this procedure to restore a previous version of Stealth(aware) and its associated database.

 Caution

  • This procedure can only be used as documented to revert to a previous version of Stealth(aware) and its associated database. Do not adapt this procedure for any other purpose, and do not attempt to manually manipulate the database, or you might permanently corrupt your environment.

  • When you perform this procedure, any changes you made to the database since the upgrade will be lost.

Do the following:

  1. Navigate to C:\Users\<user name>\AppData\Roaming\Stealth and locate the insight.db database file.

  2. Rename the database file; for example, rename it to before_restore_insight.db.

  3. Perform the procedure in Uninstalling Stealth(aware) to delete the current version of Stealth(aware).

  4. Using the ISO file that is associated with the version of the Stealth(aware) database that you want to restore, install that earlier version of Stealth(aware) software by performing the procedure in Installing Stealth(aware).

  5. Launch Stealth(aware), and verify that the only available option is Create a New Project, as shown in Stealth(aware) Start Screen.

  6. Close Stealth(aware).

  7. Navigate to C:\Users\<user name>\AppData\Roaming\Stealth and locate the insight.db database file.

  8. Rename or delete the insight.db file.

    Note: This file was created as a new Stealth(aware) database during the installation process. Because you are restoring a previous database, this new version should not be required.

  9. Do one of the following, depending on the version of Stealth(aware) that you want to restore:

    • If you want to restore a database associated with Stealth(aware) 3.2.011 or later, do the following:

      1. Locate the insight_<revision>_<yyyymmdd>_<hhmmss>.db database file in the C:\Users\<user name>\AppData\Roaming\Stealth folder.

      2. Copy this file and paste it in C:\Users\<user name>\AppData\Roaming\Stealth.

      3. Rename the file as insight.db.

    • If you want to restore a database associated with Stealth(aware) version 3.2.004 or 3.2.010, do the following:

      1. Locate the insight.db database file in the C:\Users\<user name>\AppData\Local\Stealth-<mm>-<dd>-<yyyy>_<hh>_<mm>_<ss>\app-3.2.0 folder.

      2. Copy this file and paste it in C:\Users\<user name>\AppData\Local\Stealth.

  10. Launch Stealth(aware), and verify that you see the Last Opened Project option, as shown in Stealth(aware) Start Screen.

Your database is restored.
