Modifying Solution Details (Solution Details Panel)

Use the Solution Details panel to view (and modify, if necessary) details for a specific Solution, including the Solution name and the Profiles within the Solution.

Viewing and Modifying a Solution

To view the Solution Details panel and modify the details of a Solution, do the following:

  1. From the map, select a Solution. A Solution has a blue background if it is Stealth-enabled and a white background if it is not Stealth-enabled.

    The Solution expands (if it was not already expanded), and the Solution Details panel appears.

  2. If necessary, in the Solution Name field, modify the name of the Solution. This name can be a maximum length of 28 characters and must meet the requirements that are described in Supported Characters and Length Restrictions in Enterprise Manager and Stealth(aware).

    Note: You cannot change the names of the Standard Solutions (Enterprise Manager, Infrastructure, Internet, and Intranet).

  3. Review the list of Profiles in the Profiles list.

    If you want to view the details of a Profile, click the Profile name. See Modifying Profile Details (Profile Details Panel) for more information.Modifying Node Details (Node Details Panel)

    If you want to remove a Profile from the Solution, drag-and-drop the Profile onto the map.

    Note: The Channels field appears below the Profiles field and specifies the number of Inter-Solution Channels (that is, channels which both originate and terminate within the Solution). You cannot modify or expand the Channels field. To view the details about a Channel, click the Channel on the map.

  4. If necessary, change the Stealth Enabled Added Profiles setting, which is the default setting for the Solution and its associated Profiles and Nodes. Stealth Enabled Added Profiles is set to On for all newly created Solutions.

    If you change this value, you change the default setting for the Solution. However, you can override this setting for individual Profiles within the Solution. (For example, you might have a Solution with Stealth Enabled Added Profiles set to On that includes multiple Stealth-enabled Profiles and one non-Stealth-enabled Profile.)

    Note the following:

    • If Stealth Enabled Added Profiles is set to On and you add a non-Stealth-enabled Profile to the Solution, that Profile will automatically become Stealth enabled.

    • If Stealth Enabled Added Profiles is set to Off and you add a Stealth-enabled Profile, that Profile will remain Stealth-enabled. This is because the default value for Profiles is non-Stealth-enabled, and you must have previously changed the Profile to Stealth Enabled; therefore, the value will not change automatically.

  5. Click the X in the upper-right corner to close the Solution Details panel.

    Note: When you make a change to the Solution Details panel, the change is automatically saved.

Deleting a Solution

To delete a Solution, do the following:

  1. Select the Menu icon in the upper-right corner of the Solution Details panel.

             

  2. Click Delete Solution.

  3. When prompted, click Yes, Delete.

If the Solution you are deleting includes any Profiles, those Profiles are returned to the map.
